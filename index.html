Tramontana for Processing

A Library by Pierluigi Dalla Rosa for the Processing programming environment.
Last update, 5th July 2017.

Tramontana is a platform for interactive spaces.

With Tramontana will be simple to architect choreographies and interactive spaces. Connected multiple devices to create mesmerizing immersive experiences just from a sketch.


Tramontana is available for iOS, tvOS. macOS and tramontana boards coming soon.
You can download Tramontana
- for iOS here.
 - for macOS here.

Download

Download Tramontana version 0.9.5 in .zip format.

Installation

Unzip and put the extracted tramotana_processing folder into the libraries folder of your Processing sketches. Reference and examples are included in the tramotana_processing folder.

Keywords. interactive spaces - complex system - state machines - prototyping - choreography

Reference. Have a look at the javadoc reference here. A copy of the reference is included in the .zip as well.

Source. The source code of Tramontana is available on github, and its repository can be browsed here.
